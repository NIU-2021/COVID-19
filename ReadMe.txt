Association of anti-contagion policies - 2021

General info
This is a repository contains cleaned code (Model.py) used to produce results for the paper: "Association of anti-contagion policies with the spread of COVID-19 in the Unites States"
To perform the code, Python (version 3 and higher) is used.
On average running the code for each state on a "normal" desktop computer would take about 5-15 minutes.

Additional info
All input files for every state of the U.S. are available on a repository file named "Input_Files". 
Also, data frames for every state representing the effective duration of each implemented policies for all the U.S. states are available on a repository file named "Additional_Files"  


Method description
To follow the methodology please check the methods section in the manuscript.
